# Product Overview

Lingo is a language learning chat application that reimagines Yahoo! Chat with modern AI capabilities. It combines nostalgic early-2000s aesthetics (Windows XP/Yahoo Messenger style) with real-time chat and AI-powered language assistance.

## Core Features

- Real-time chat rooms for language practice (Firebase Realtime Database)
- AI Language Buddy for grammar help, translations, and pronunciation tips (Google Gemini)
- Draggable/resizable windows in a desktop environment
- Voice input for speech-to-text practice
- Multiple nostalgic themes (Retro Yahoo, Halloween, Cyberpunk, Clean Modern)
- User presence tracking and keyboard shortcuts

## Target Users

- Language learners practicing writing and conversation
- Language teachers monitoring and providing feedback
- Language exchange partners connecting globally

## Authentication

Currently uses Firebase Authentication with email/password and Google sign-in. Users must be authenticated to access the desktop environment.
